How do I find it?

  1. Using the Advanced Library Search select “Title” under the “Any field” drop-down
  2. If searching for a journal, navigate to the article by searching for the title in double quotation marks, then locate the article using the year, volume and issue details.
  3. Use Google Scholar and search for the full reference. Tip: make sure you have set up the Library links so you can access full-text articles/items.
